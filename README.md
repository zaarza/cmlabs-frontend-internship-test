## Installation

### Development Mode

1. Clone this repository
2. Installed required packages

```
  cd cmlabs-frontend-internship-test
  npm install
```

3. Run project by

```
  npm run tailwind
```

4. Run with live server VSCode extensions

### Production Mode

1. Clone this repository
2. Installed required packages

```
  cd cmlabs-frontend-internship-test
  npm install
```

3. Copy the folder and paste into your web server directory
